## 1.Project Title
# Customer-Churn-Prediction
## 2.Problem Statement
This project aims to Predict whether a telecom customer is likely to churn based on historical customer data.The goal is to help improve customer retention strategies. 
## 3.Dataset
Dataset used : Telco Customer Churn Dataset(Kaggle)
It contains customer demographics, account information, and service usage details.
## 4.Workflow
The project includes the following steps:
- Exploratory Data Analysis (EDA)
- Data preprocessing
- Feature scaling
- Model building
- Model evaluation
## 5.Models Used
The following machine learning models were used:
- Logistic Regression
- Random Forest
- LightGBM
## 6.Results
Model performance:
- Logistic Regression: 81.54%
- Random Forest: 80.97%
- LightGBM: 80.76%
## 7.Conclusion
Logistic Regression performed slightly better in this case. The model can be used to identify customers at risk of churn and support retention strategies.
